# Extras

Beyond managing AI coding agents, AoE has a few fun extras built in.

## settl -- Play Settlers of Catan

[settl](https://github.com/mozilla-ai/settl) is a terminal Settlers of Catan game with AI opponents, built with ratatui. Select **settl** from the tool picker when creating a new session to launch a game right inside AoE.

Install settl ([instructions](https://github.com/mozilla-ai/settl)), then pick it as the tool in the new-session dialog. AoE integrates with settl's hook system so game status (Running, Waiting, Idle) shows up in the dashboard automatically.

## Sound Effects

AoE can play RPG sound effects when agent sessions change state. Hear a spell cast when a session starts, a blade swing when an agent is working, or a roar when something errors out.

See the full [Sound Effects](/docs/sounds/) guide for setup, configuration, and custom sound support.
